# 🚀 SmartLearn – AI-Based Academic Support Platform

SmartLearn is a modern AI-powered web application that helps students study smarter, faster, and more efficiently using PDF documents.

It combines document reading, AI assistance, and revision tools into one unified platform — eliminating the need for multiple apps.

---

## 📌 Overview

Students today rely heavily on digital learning materials like PDFs, slides, and notes. However:

- 📄 Reading long documents is time-consuming  
- 🔍 Finding key concepts is difficult  
- 🧩 Learning tools are scattered across different apps  

👉 SmartLearn solves this by integrating everything into one platform.

According to the system workflow, users upload documents and AI processes them to generate summaries, quizzes, flashcards, and answers.

## 🏗️ System Architecture

SmartLearn follows a **3-tier architecture**:

- 🎯 Frontend (React + Tailwind CSS)  
- ⚙️ Backend (Node.js + Express)  
- 🗄️ Database (MongoDB)  
- 🤖 AI Layer (Google Gemini API)  

The architecture diagram shows how data flows from user → frontend → backend → AI/database → frontend. 

---

## 🔄 Workflow

1. User registers/logs in  
2. Uploads PDF document  
3. System extracts text  
4. AI processes content  
5. User can:
   - Ask questions  
   - Generate summaries  
   - Create flashcards  
   - Generate quizzes  
6. Results stored & shown in dashboard
